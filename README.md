# HomeRunGame

制作期間：2020/02/21 ~ 2020/03/13

・使用したツール・ライブラリ
  VisualStudio2017
  DXライブラリ(Android ver)
  XAMPP(Appache, MySQL)
  
・概要
タイミングを合わせてタップし、打球のパワーと角度を決めると、スコアが算出されるAndroid用のゲーム。
ネットワーク(今回はXAMPP)を使用すると、スコアがランキング形式で取得できる。
